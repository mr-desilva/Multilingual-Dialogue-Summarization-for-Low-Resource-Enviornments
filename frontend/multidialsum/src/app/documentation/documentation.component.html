Fine Tuned bart-large-xsum model

This model is fine tuned using TWEETSUM dataset. Which contains 1100 dialogues and summary pairs which are annotated by humans. Model is trained using the huggingface library.

Cross-lingual Transfer Model m2m100

This model is a many to many model which can translate language between multiple languages. This model supports 97 languages, but the performance on the each language may not be the same due to it's different amount of training data in each language.

Evaluation and Testing

bart-large-xsum based dialogue summarization model is tested using rouge score, The initial dataset were split into training, testing and evaluation data. Fine tuned model is tested using the evalution data. Evaluation on the entire system with multiple dialogues were done using human evaluation method. Including english german, spanish and french dialogue-summary pairs were included in a survey. Participants for the survey is selected from social media platforms where people are fluent with certain language specially from language learning spaces.
