About The Project

The aim of this research is to design, develop and evaluate a multilingual dialogue summary generation system for customer services using dialogue summarization model with the help of the cross-lingual transfer method.

Technologies

Use of pre trained language transformer - bart-large-xsum for developing the dialogue summarization model using huggingface library. To achieve the multilingual or multi language a cross lingual transfer model which is the m2m100 model was selected. Together these two models the pipeline of dialogue summarization which can support more than one language was created.
